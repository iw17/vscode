# 须知
## 依赖环境
- Node.js
- pnpm
## 依赖
- [Nextra](https://nextra.site/)及其附带的
- [Nextra-docs-theme](https://nextra.site/docs/docs-theme/start)
- [Next.js](https://nextjs.org)
## 内容组织
内容都在[pages](./pages)目录下，使用markdown和[_meta.json](./pages/_meta.json),也可以使用mdx。

pnpm build

pnpm preview
# 开发
pnpm dev
